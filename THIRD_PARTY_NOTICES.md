# Third-party acknowledgements

This standalone project grew out of an earlier prototype built alongside
[phuryn/claude-usage](https://github.com/phuryn/claude-usage). Paweł Huryn's
project demonstrated a practical way to inspect Claude Code JSONL transcripts
and is distributed under the MIT License. The standalone collector in this
repository was implemented specifically for the terminal graphs and does not
bundle the upstream web dashboard, VS Code extension, or scanner. The original
project remains credited here to preserve the development provenance.

The terminal presentation is visually inspired by the token activity screen in
OpenAI Codex. No Codex source code or assets are included.
